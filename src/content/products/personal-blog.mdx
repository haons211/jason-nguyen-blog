This personal blog showcases modern web development practices using Next.js 15 with the App Router architecture. It features a complete content management system using MDX files, advanced search functionality with Fuse.js, and a fully responsive design optimized for all devices.

## Key Features

- **Modern Architecture**: Built with Next.js 15 App Router for optimal performance
- **Content Management**: MDX-based content system for easy writing and maintenance
- **Advanced Search**: Fuzzy search functionality powered by Fuse.js
- **Responsive Design**: Optimized for all devices and screen sizes
- **SEO Optimized**: Built-in SEO features and metadata management
- **Performance**: Optimized for Core Web Vitals and fast loading times

## Technical Highlights

The blog includes features like table of contents generation, syntax highlighting for code blocks, and a clean, professional design that focuses on readability and user experience. The architecture follows modern React patterns with Server Components where possible for optimal performance.
